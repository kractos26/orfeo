 Requerimientos iniciales para la implementación del módulo de Administración
Martha Mera

1. Descripción

Permite realizar la parametrización de las diferentes formas de envio de correspondencia al exterior de la entidad.

2. Actores

2.1 Usuario normal

Usuario administrador del sistema

3. Flujo de Eventos

3.1. Flujo Básico ó Normal : Crear Forma de Envio

3.1.1 Precondiciones

3.1.1.1. El actor debe haber ingresado al sistema
3.1.1.2. El usuario tiene permiso de administrador del sistema
3.1.1.3. El usuario ha seleccionado el link "Administracion"  en la lista de permisos especiales


1. A: INICIO - Selecciona el link "Administracion"  en la lista de permisos especiales

 



2. S: Ingresa al menú principal del módulo de Administración.

3. A: Selecciona la opción "3. FORMAS DE ENVIO"

4. S: Despliega la siguiente ventana con la siguiente forma:




4.1 En la parte superior una caja de selección rotulada "Seleccione Codigo", aqui aparece la lista de los códigos de las formas de envio que se encuentran actualmente en el sistema.
4.2 Caja de texto rotulada "Ingrese código" (Campo numérico)
4.3 Caja de texto rotulada "Ingrese Forma de Envío" (Campo Alfanumérico)
4.4 Selección con radio boton rotulado "Seleccione estado" con las opciones de "Activado" y "Desactivado", se encuentra seleccionado "Activado"
4.5 Un botón rotulado "Listado"
4.6 Un botón "Agregar"
4.7 Un botón "Modificar"
4.8 Un botón "Eliminar"


5. A: Diligencia los datos en la sección "Ingrese Código" e "Ingrese Forma de Envío", selecciona el botón "Agregar"

6. S: Confirma la creación del nuevo tipo de Forma de Envio. Este ya aparece en la lista de "Seleccione Código" - FIN




3.2. Flujo Básico ó Normal: Modificar Forma de Envio

3.2.1 Precondiciones

3.2.1.1. El actor debe haber ingresado al sistema
3.2.1.2. El usuario tiene permiso de administrador del sistema
3.2.1.3. El usuario ha seleccionado el link "Administracion"  en la lista de permisos especiales
3.2.1.4. El usuario ha seleccionado el link "Formas de Envio"
3.2.1.5 Ya existen creadas en el sistema "Formas de Envio"

1. INICIO: A: Selecciona el código que desea modificar
2. S: Despliega la información correspondiente a esta forma de envio visualizandolo en el campo de Código y Forma de Envío. El campo de código se encuentra deshabilitado y no se puede cambiar.
3. A: Realiza los cambios  en  la descripción de "Forma de envio" y/o "Estado".  Da click en el botón "Modificar"
4. S: Realiza el cambio en las bases de datos respectivas actualizando la información. Envia la confirmación del cambio de datos: "Forma de envio modificada satisfactoriamente"
 - FIN


3.3. Flujo Básico ó Normal: Eliminar Forma de Envio

3.3.1 Precondiciones

3.3.1.1. El actor debe haber ingresado al sistema
3.3.1.2. El usuario tiene permiso de administrador del sistema
3.3.1.3. El usuario ha seleccionado el link "Administracion"  en la lista de permisos especiales
3.3.1.4. El usuario ha seleccionado el link "Formas de Envio"
3.3.1.5 Ya existen creadas en el sistema "Formas de Envio"

1. INICIO: A: Selecciona el código que desea eliminar
2. S: Despliega la información correspondiente a esta forma de envio visualizandolo en el campo de Código y Forma de Envío. El campo de código se encuentra deshabilitado y no se puede cambiar.
3. A: Da click en el botón "Eliminar"
4. S: Verifica que en el sistema no se ha realizado ningún tipo de envio teniendo en cuenta la "Forma de Envio" escogida para eliminar. Realiza el cambio en las bases de datos respectivas actualizando la información
Envia la confirmación de la eliminación de datos: "Forma de envio eliminada satisfactoriamente". - FIN


3.4. Flujo Básico ó Normal: Ver listado Formas de Envio

3.4.1 Precondiciones

3.4.1.1. El actor debe haber ingresado al sistema
3.4.1.2. El usuario tiene permiso de administrador del sistema
3.4.1.3. El usuario ha seleccionado el link "Administracion"  en la lista de permisos especiales
3.4.1.4. El usuario ha seleccionado el link "Formas de Envio"
3.4.1.5 Ya existen creadas en el sistema "Formas de Envio"

1. INICIO: A: Selecciona el botón "Listado"
2. S: Despliega la información correspondiente a las formas de envio que existen creadas en el sistema.
- FIN
